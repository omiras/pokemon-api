Documentación de la API PokeAda

/api/v1/pokemon/:pokemonName (GET)

Obtener información sobre un Pokémon específico por nombre.


GET /api/v1/pokemon/:pokemonName

Ejemplo de invocación:

/api/v1/pokemon/charizard

/api/v1/pokemon-species/:id (GET)

Obtener información sobre una especie de Pokémon específica por ID.


GET /api/v1/pokemon-species/:id

Ejemplo de invocación:

/api/v1/pokemon-species/25

/api/v1/pokemon (GET)

Obtener una lista de Pokémon con parámetros de paginación opcionales.


GET /api/v1/pokemon

Ejemplo de invocación:

/api/v1/pokemon?limit=10&offset=0
